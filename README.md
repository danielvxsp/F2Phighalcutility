a project to help with high alcing in osrs that i had unfinished for a while. 
i finished it up with some help from gemini so ill be cleaning the AI slop up as i develop it more

very similar to the alc utility from the wiki but i saw some things lacking so 
i wanted to make my own version to build off and add what i felt should be avaliable

notably you can mark items as bought to start the GE limit cooldown timer
this timer starts from when the first item in your order is filled so 
you should mark it to start the timer right after setting the order

for now this only supports F2P items since i dont see myself looking for items 
to highalc once i get members any time soon and its a lot easier to handle with the smaller item pool
